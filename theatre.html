Chernivtsi theatre website

A website for my hometown theatre.

A second multipage website for my hometown theatre built with Next.js. The project has google authentication, integrated Stripe payment for tickets (press "Buy ticket" button to test), mail subscription form connected to mailchimp which will send you welcome email if you subscribe. The website is fully responsive and styled with TailwindCSS.
Link to github.

Techonolgies and libraries used:

  • Next.js
  • Stripe API
  • Mailchimp
  • TailwindCSS
  • Keen-slider
  • Next-auth
